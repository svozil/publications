An article by Popescu and Rohrlich \cite{popescu-97b}
contains the negative correlation function of Eq.~(1).
These authors mention that, for ${a} =\pi /2 $ ,${a}'=0$, ${b} =\pi /4$, ${b}'= 3\pi /4$,
the Clauser-Horne-Shimony-Holt (CHSH) inequality
$
\vert
E({ a} ,{ b} )+
E({ a} ,{ b} ' )+
E({ a}' ,{ b} )-
E({a} ',{ b} ')
\vert
\le 2
$
is violated with the maximal algebraic value 4.
This explicit example answers the question whether memoryless single bit exchange protocols can give rise
to maximal violations of the CHSH inequality to the positive.


The author acknowledges Akbar Fahmi for pointing out this result.
